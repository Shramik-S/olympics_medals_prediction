# Olympic Medals Predictor

Forecasts future medal tallies by country using past performance, athlete age, and participation data.

**@shramik_s** | Nov 2025

---

## Data
- Historical medals  
- Average athlete age  
- Number of participants  

---

## Model
- ML-based (regression)  
- Trained on recent Olympic cycles  
- Focus: top-performing nations  

---

```bash
git clone https://github.com/Shramik-S/olympic-predictor
